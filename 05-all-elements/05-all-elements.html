All elements

<a>

The anchor element (<a>) is one of the fundamental elements in HTML.

It allows a user to click on the link text it provides, and navigates to that link's location.

<abbr>

The element (<abbr>) represents an abbreviation or acronym.

This is used along with a title attribute to associate a full-text explanation with an abbreviation or acronym.

<article>

The element(<article) is used to represent a self-contained in adocument or article.

<b>

The bold element (<b>) is one of the html elements.

This is used to bold words.

<button>

The button element (<button>) is one of the html elements.

It is useful when making a table to make a form.

<body>

We have to use this element (<body>) in html because this contains all of the contents in our code or documents

<br>

The break element (<br>) is very useful when writing paragraph.

It produces a line in text.

<code>

The HTML (<code>) element displays its contents styled in a fashion intended to indicate that the text is a short fragment of computer code.

<dd>

The element(<dd>) provides the description, definition, or value for the preceding term in a description list.

<dl>

The HTML element(<dl>) represents a description list.

Common uses for this element are to implement a glossary or to display metadata (a list of key-value pairs).

<dt>

The element(<dt>) specifies a term in a description or definition list, and as such must be used inside a (<dl>) element.

It is usually followed by a (<dd>) element.

<data>

The HTML (<data> element links a given content with a machine-readable translation. If the content is time- or date-related, the (<time> element must be used.

<div>

The HTML element (<div>) defines a division or a section in an HTML document.

It is often used as a container for other HTML elements to style them with CSS.

<footer>

The element(<footer>) is used to context the information.

<head>

The element(<head>) is a container for all data and is placed between the (<html>) tag and the (<body) tag.

<h1>

The element(<h1>) is used to make a big title.

<h2>

The element(<h2>) is useful when making subheadings or sections.

<h3>

The HTML element(<h3>) is used to write a samall headings inside subheadings.

<h4>

The HTML element(<h4>)represents a level 3 heading.

<h5>

The HTML element(<h5>) defines a section heading of level five.

<h6>

The HTML element(<h6>)

<hidden>

The hidden element(<hidden>) attribute is a boolean attribute.

It specifies that an element is not yet, or is no longer, relevant.

<header>

The HTML element(<header>) is intended to usually contain the section's heading (an h1-h6 element).

<iframe>

The HTML element (<iframe>) is used to nest webpages.

Also, a whole webpage can be contained in an iframe.

<img>

The HTML element (<img>) is used very often in html.

It is a way of displaying a static image as a content on a page.

<li>

The element (<li>) helps us making list of documents or names.

<nav>

The HTML element (<nav>) defines a set of navigation links.

<ol>

We use the element (<ol>) when we are making ordered lists.

<p>

The element(<p>) is one of the basic elements in HTML.

We use it when we are writing a paragraph in a document.

<script>

The element(<script>) points to an external script file through the src attribute.

<style>

The element(<style>) is used for making styles in a table.

<table>

The (<table>) tag defines an HTML table.

<tbody>

The element(<tbody>) is used in conjunction with the (<thead>) and (<tfoot>) elements to specify each part of a table.

<thead>

The element(<thead>) is the same as the (tbody).

<tfoot>

The element(<tfoot>) defines the end in a table.

<title>

The title element(<title>) defines the document's title that is shown in a browser's title bar or a page's tab.

<tr>

The table row element (<tr>) aligns the child cells in row when we create a table.

<th>

The HTML element (<th>) is a table heading.

It is used to make headings inside a table.

<ul>

This element is used to make bulleted lists.
